# Lista Zakupów - aplikacja webowa

## Opis projektu

Aplikacja „Lista zakupów” to aplikacja webowa umożliwiająca zarządzanie listą zakupów. Użytkownik może dodawać, edytować, usuwać oraz oznaczać produkty jako kupione. Dane są zapisywane lokalnie w przeglądarce. Dodatkowo aplikacja pobiera ciekawostki z zewnętrznych API.

---

## Wymagania funkcjonalne

1. Użytkownik może dodawać produkty do listy zakupów.
2. Użytkownik może usuwać produkty z listy.
3. Użytkownik może edytować nazwę produktu.
4. Użytkownik może oznaczać produkty jako kupione.
5. Użytkownik może filtrować produkty (wszystkie, kupione, niekupione).
6. Użytkownik może przeglądać dane pobierane z zewnętrznego API.

---

## Wymagania niefunkcjonalne

1. Aplikacja jest responsywna i działa poprawnie na urządzeniach mobilnych.
2. Dane użytkownika są zapisywane lokalnie w przeglądarce (localStorage).

---

## Potencjalni odbiorcy

1. Osoby robiące codzienne zakupy.
2. Studenci i osoby mieszkające samodzielnie.

---

## Korzyści dla użytkowników

1. Lepsza organizacja zakupów.
2. Brak konieczności zapamiętywania listy produktów.

---

## Stos technologiczny

* HTML
* CSS
* JavaScript
* REST API (zewnętrzne serwisy z ciekawostkami)
